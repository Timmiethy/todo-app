# Project Tech Stack: Todo Practice App

This document outlines the tech stack used for this todo app mini-project, along with the overall design and casual updates of the progess.
---

## 1. Frontend
*   **Framework:** **React 19** (via **Vite**)
*   **Language:** **TypeScript**

## 2. Styling
*   **Utility-First CSS:** **Tailwind CSS 4.0**

## 3. State Management
*   **API State:** **TanStack Query** (React Query)
*   **Global App State:** **Zustand**

## 4. Backend
*   **Framework:** **NestJS**

## 5. Database
*   **Database:** **MongoDB**
*   **ODM:** **Mongoose**
